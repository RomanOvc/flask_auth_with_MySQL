# flask_auth_with_MySQL

In terminal

Step 1:

for MySQL: 
1) flask db init
2) flask db migrate
3) flask db upgrade



Step 2:
for Windows:
1) set FLASK_APP=flasky.py
2) set FLASK_DEBUG=1
3) set FLASK_ENV=development

Step 3:
1) set MAIL_USERNAME=myusername@gmail.com
2) set MAIL_PASSWORD=youpassword

Step 4:
  flask run
